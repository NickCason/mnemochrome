# Mnemochrome

> See a color, then summon it back.

## About

Mnemochrome is a mobile-first PWA color-memory game. Players see a target color full-screen for a configurable number of seconds and must recreate it using a touch-based color picker. The guess is scored using ΔE2000 (perceptually-uniform color distance), and the score is rendered as a percentage. Endless rounds are available, tracking personal bests and the last 10 averages.

## Play

[https://nickcason.github.io/mnemochrome/](https://nickcason.github.io/mnemochrome/)

## Local development

```bash
npm install
npm run dev
```

Note that the dev server binds to 0.0.0.0:5173, making it accessible from other devices on your local network (or tailnet).

## Tech

- Vite + TypeScript (no UI framework)
- vite-plugin-pwa (installable PWA, offline)
- culori (perceptual color math)
- sharp (build-time icon generation)
- Vitest (tests)
